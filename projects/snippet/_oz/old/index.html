Default Colors on Bootstrap

Changing default colors of Bootstrap without using Sass preprocessor (produces scss).

This alert is set to alert-primary, which should be blue. However, due to my custom change-default-colors.css, it is now purple!
View change-default-colors.css

Note: In the current version of Bootstrap, CSS variables are set after Sass compiles the CSS. This means you cannot change all of the default colors easily through a small set of variables in root. Since each class or element color would need to be changed manually, Sass is the ideal way to change these default colors.
